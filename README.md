# Rock-Paper-Scissors Game

## Purpose
Simple C++ console game: player vs computer.

## Prerequisites
- C++ compiler (g++ or clang++)  
- VS Code (optional)

## How to Play
- Enter `r` → Rock, `p` → Paper, `s` → Scissors, `q` → Quit  
- Computer chooses randomly, winner displayed, scores updated
